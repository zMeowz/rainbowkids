# 👟 Rainbowkids — Дитячий інтернет-магазин взуття

Сучасний статичний сайт інтернет-магазину дитячого взуття. Готовий для розгортання на GitHub Pages.

## 🚀 Запуск на GitHub Pages

1. Завантажте всі файли в репозиторій GitHub
2. Перейдіть у **Settings → Pages**
3. У розділі **Source** оберіть гілку `main` та папку `/ (root)`
4. Натисніть **Save**
5. Сайт буде доступний за адресою: `https://YOUR-USERNAME.github.io/REPO-NAME/`

## 📁 Структура файлів

```
rainbowkids/
├── index.html              ← Головна сторінка
├── css/
│   └── main.css            ← Вся стилізація (дизайн-система)
├── js/
│   └── main.js             ← Вся інтерактивність
├── pages/
│   ├── catalog.html        ← Каталог товарів з фільтрами
│   ├── product.html        ← Сторінка товару
│   ├── checkout.html       ← Оформлення замовлення
│   ├── delivery.html       ← Доставка і оплата
│   ├── returns.html        ← Повернення та обмін
│   └── 404.html            ← Сторінка 404
└── README.md
```

## 🔮 Наступні кроки (Production)

Для повноцінного магазину рекомендуємо:

**Варіант 1: Next.js + Sanity CMS** *(рекомендовано)*
- Швидкий, SEO-friendly
- Зручна адмінка для товарів
- API для замовлень

**Варіант 2: WordPress + WooCommerce**
- Знайома адмінка
- Багато плагінів
- Хостинг: від 100 ₴/міс

**Інтеграції для production:**
- Оплата: LiqPay, Wayforpay, Monobank
- Доставка: Nova Poshta API
- Аналітика: Google Analytics 4, Meta Pixel
- CRM: KeyCRM або NetHunt
